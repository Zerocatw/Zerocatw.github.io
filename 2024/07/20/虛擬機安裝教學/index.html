虛擬機安裝&kali

Zerocatw
  2024-07-20 05:54:18 2024-07-20 05:54:18 Created  
  • 安裝 
 
  • VMware 

前言

以下安裝環境作業系統都為Windows為主

總共有兩款虛擬機
VirtualBox&Vmware workstation pro 17
前者簡單好操作
後者性能較好
都可用

注意注意
建議筆電/主機要有16GB RAM 不然會當掉
可用工作管理員查看，記憶體那邊至少要??/16GB
因為需要分配記憶體給虛擬機用

VirtualBox

安裝VirtualBox

https://www.virtualbox.org/wiki/Downloads
選擇Windows hosts

下載後打開，點下一步

看你要安裝在哪，預設是安裝在c槽

點是

如果出現這個，一樣點是

點安裝

安裝完成

Vmware workstation pro 17

安裝Vmware workstation pro 17

因為VMware被收購了，直接開放免費下載，但是需要先註冊帳號

訪問網址 : https://support.broadcom.com/

點右上角的Register註冊帳號

註冊好後點Login

成功登入後點選 Software -> VMware Cloud Foundation -> My Downloads

會看到一堆VMware的產品
往下滑找到VMware Workstation Pro 點進去

我們這裡下載Personal Use (Windows) 17.5.2最新版

接下來可能會要你填資料，填一填後按送出，只填這一次，之後下載其他的咚咚就不用再填惹
因為我已經填過了，就沒有放截圖惹

好了之後按下小雲朵就可以下載惹!

載好之後打開，點Next

打勾後點Next

如果電腦有開Hyper-V那可能會看到這個畫面，可以打勾後按Next就好

選擇你要安裝的位置，預設是C槽，並把Enhance keyboard打勾，點Next

這裡第一個是自動更新，第二個是是否要加入用戶改善計畫，可以看個人要不要勾選，好了之後點Next

點Next

點Install

接下來重新開機一下
完成之後打開VMware Workstation，直接Continue就好

成功打開

安裝kali

https://www.kali.org/get-kali/#kali-virtual-machines

如果剛剛的虛擬機，用的是VirtualBox，那就載第二個，是VMware的話，就載第一個

VirtualBox

下載下來解壓縮後進到資料夾

點兩下藍色的那個(kali-linux-2024.1-virtualbox-amd64.vbox)
便會直接幫你開啟VirtualBox

之後案上方的「啟動」，跑一段時間後會到這裡

輸入帳密(預設)
username : kali
password : kali

安裝成功

VMware Workstation

下載下來解壓縮後
打開VMware Workstation

點中間Open a Virtual Machine

找到剛剛那個資料夾點開，選擇(kali-linux-2024.1-vmware-amd64.vmx)

接下來按Power on即可打開

成功進入到登入畫面

輸入預設帳號密碼
username : kali
password : kali

成功登入
